## [1.0.0] - 2026-07-22

### 🚀 首次发布

#### GPX 轨迹渲染
- 支持 GPX 文件上传（点击 + 拖拽）
- 地图自动缩放适应轨迹范围
- 起点（S）/ 终点（E）标记
- 街道 / 卫星底图切换（地图右上角）
- 右下角实时显示鼠标经纬度

#### 📷 机位踩点 (POI)
- 在地图上标记拍摄点 / 兴趣点
- 6 种 POI 类型：📸 拍照点 / 🚻 厕所 / 🪑 休息 / 🏪 补给点 / 💧 饮水 / ⚠️ 注意
- 点击 POI 可删除
- POI 数据 JSON 导出 / 导入
- 工具栏位于地图左上角（工具栏 / 面板内）
- POI 自动持久化到 localStorage

#### ⛰️ 海拔剖面
- Canvas 绘制海拔图
- 距离刻度（km）
- 海拔刻度（m）
- 三列布局（距离 | 海拔 | 爬升）

#### 🚇 隧道过滤
- 基于梯度检测的 GPS 异常点过滤
- 可调坡度阈值（5%-30%）和平滑窗口（0-100）
- 可开关过滤功能
- 实时显示过滤了多少无效爬升

#### 🚴 电子骑友
- 沿轨迹匀速前进动画
- 默认 25 km/h，可调速度（5-50 km/h）
- 进度条 + 百分比
- 预估到达时间
- 起点 / 终点 / 当前位置标记（S / E / 箭头）
- 暂停 / 重置控制

#### 📊 数据统计
- 距离 (km)
- 爬升 (m)
- 时长（按 25km/h 匀速计算）

#### 🔗 路书链接
- Strava
- iGPSPORT
- Garmin Connect

#### 📤 分享
- 微博分享
- Twitter 分享
- 复制链接
- 活动标题自动设置为 GPX 文件名

#### 💾 数据持久化
- 用户名保存到 localStorage
- POI 数据保存到 localStorage
- 页面加载自动恢复上次状态
